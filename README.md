#  ProyectoAndroid - Pokédex Digital

>  Proyecto educativo desarrollado como caso práctico de aprendizaje en Android Studio.

## ¿De qué trata?
App Android que simula una Pokédex digital, desarrollada para aprender los fundamentos del desarrollo móvil.

##  Aviso
Este proyecto está hecho con fines de **aprendizaje**, no está pensado para uso en producción ni para su descarga y reutilización directa. Se comparte únicamente para su consulta y revisión. Contiene los primeros 151 pokémon con la información básica oficial en tiempo real.
